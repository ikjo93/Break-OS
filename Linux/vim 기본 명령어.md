+ vi {파일명} : 해당 파일을 연다
  + vi -R {파일명} : 해당 파일을 읽기 모드로 연다 (= view file)

<br>

+ ESC : 명령(일반) 모드 전환

<br>

+ i : 편집 모드 전환

<br>

+ ESC + :w + Enter : 편집기를 종료하지 않고 파일을 저장함

<br>

+ ESC + :w + 변경할 파일명 + Enter : 편집기를 종료하지 않고 파일을 다른 이름으로 저장함

<br>

+ :wq : 파일을 저장하고 편집기를 종료함 (= :x, ZZ)

<br>

+ :q! : 파일을 저장하지 않고 편집기를 종료함
